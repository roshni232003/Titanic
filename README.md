I worked on a classic Data Science project using the Titanic Dataset to build a predictive model that determines whether a passenger survived the Titanic disaster.

The dataset included several passenger attributes such as:


• Age

• Gender

• Passenger Class

• Fare

• Cabin

• Survival status

By exploring and analyzing this data, I built a model that learns patterns and predicts survival probability.
Through this project, I strengthened my understanding of:
✔ Data Cleaning & Preprocessing
✔ Exploratory Data Analysis (EDA)
✔ Feature Analysis
✔ Predictive Modeling
